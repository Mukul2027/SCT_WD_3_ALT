# SCT_WD_3_ALT - Tic-Tac-Toe Web Application

This project was developed as part of the SkillCraft Technology Web Development Internship.

## Features

- Two Player Gameplay (X and O)
- Winner Detection
- Draw Detection
- Game Restart Functionality
- Interactive User Interface
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Task Objective

Build a Tic-Tac-Toe web application that handles user interactions, tracks game state, checks winning conditions, and provides an engaging gameplay experience.
